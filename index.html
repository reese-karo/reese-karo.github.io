GitHub

Predicting Car Auction Sales

GitHub

In this project, I forecasted car auction selling prices using machine learning algorithms, including Gradient Boosted Trees, Random Forests, Elastic-Net Regression, and Neural Networks. I conducted extensive exploratory data analysis, constructed pipelines for feature preprocessing, and tuned models using cross-validation and grid search. The final Gradient Boosted Tree model achieved an RMSE of 4095 on training data, with age, odometer, and condition identified as the most influential predictors.
