Additional Projects

Below are some of the projects I have worked on during my time at UCSB, including data analysis and building multiple machine learning models for predicting car auction sales, and modeling a European call option using numpy.

Predicting Car Auction Sales

GitHub Final Report

In this project, I forecasted car auction selling prices from a Kaggle dataset consisting of over 100,000 car sales,and by using machine learning algorithms, including Gradient Boosted Trees, Random Forests, Elastic-Net Regression, and Neural Networks. I conducted extensive exploratory data analysis, constructed pipelines for feature preprocessing, and tuned models using cross-validation and grid search. The best performing model was a Deep neural network with a testing RMSE of 2380, achiveing better results than the Xgboosted model.

European Call Option Pricing

GitHub

In this project, I modeled a European call option using numpy. I used the Black-Scholes formula to calculate the theoretical price of the option and then compared it to the price calculated using the binomial model. I also used the binomial model to calculate the delta of the option.
